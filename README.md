# Simple Music Player

A lightweight desktop music player built with Python + PyQt6.
Browse your library and play audio with a simple interface.

## Requirements
- Python 3.10+
- pip
- Git (optional, for cloning)

## Setup
1. Clone or download the project, then open terminal/command-prompt in the project folder (the folder containing simple-music-player.py).

You should see:
```
simple-music-player.py
icons/
requirements.txt
run.sh
run.bat
```

### Linux (Kubuntu / Ubuntu / Debian)
1. Install venv support (one time)
```
sudo apt install python3-venv
```

2. Create virtual environment
```
python3 -m venv venv
```

3. Activate it
```
source venv/bin/activate
```

4. change directory up twice, back into the main folder
```
cd ../..
```

5. Install dependencies
```
pip install -r requirements.txt
```

6. Double-click launch.sh

### Windows
1. Create virtual environment
```
python -m venv venv
```

2. Change to Scripts directory
```
cd venv\Scripts
```

3. Activate the virtual environment
```
activate
```

4. Change to main directory
```
cd ../..
```

5. Install dependencies
```
pip install -r requirements.txt
```

6. Run launch.bat
